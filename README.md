## Build

- install ubuntu-16.04.2-desktop-amd64
- run commands:

```
# sudo apt-get update

# mkdir work
# cd work
# git clone https://github.com/cbhust/STM32F429_Discovery_FreeRTOS_9.git
# cd STM32F429_Discovery_FreeRTOS_9/Projects/Hello_RTOS/
# make
```
